        CSS로 꾸미기 설정을 진행할 때는 5가지 방법이 존재
        1. 태그 옆에 style="원하는 스타일 작성"

        2. head 태그 안에 style 태그를 작성해서 스타일 적용
            [1] : 태그 이름을 작성한 후 스타일 적용
                
                
내가 왜 그랬을까

            [2] : 태그에 class 지정해준 후 class에 대한 스타일 적용
            
            [3] : 태그 옆에 id="변수명" 설정한 다음 
            style 태그 내 변수명에 대한 스타일 적용. 앞에 '.'이 아니라 '#'을 붙인다.
            

                
                abc에 대한 명시
            

            id는 javascript에서 많이 쓴다.
            style 관련 >> class 사용 권장 (여러 태그에 style 적용사 용이)
            javascript >> id 사용 권장

            [4] 태그에 작성된 특정 속성을 사용해 선택
            
             << 특정 name에 대한 style
             << 모든 name에 대한 style

        
        3. ".css" 형식 파일을 사용해서 css 파일 안에 css 코딩을 작성해주는 방법.
        이 방법을 통해 중복되는 style sheet 불러올 수 있음
        head 태그 안에 style 대신 link 태그 사용해서 작성
        >> 외부 css 파일과 연결하는 태그
        rel(relationship) : 연결하려는 파일이 어떤 종류안지?
        href(hyper reference) : 참조하려는 파일의 위치
        * 이러한 기능은
        html 내 코드길이 축소
        중복되는 코드를 하나의 파일에 작성후 불러올 수 있게 함
