레이아웃 (layout)

사용 목적을 기반으로 시각적 효과를 구성하는 행위 (어떻게 예쁘게 꾸밀 수 있나?)
사전적 의미 : 배치하다 / 정리하다 기술적 의미 : 구성요소를 제한된 공간에 효율적으로 배치하는 것

디스플레이(Display) :

화면 표시나 형식에 대한 배치 지정
태그가 화면에 어떻게 보여질지 형식을 지정하는 속성 

            - block ..화면을 수직 분할 (한 줄 다 차지하겠다, 행을 나눔)
                +)width, height 지정 가능
                
            - inline : 화면을 수평 분할(디자인을 글자, 문자열 취급)
                +)가로, 세로 지정 불가능
                
            - inline-block : inline 처럼 수평분할은 가능하되, block 처럼 가로 세로 지정가능
                +)가로, 세로 지정 가능

            - none : 태그가 존재하지만 화면에는 보이지 않게 숨김

            - flex : 태그 간 간격, 정렬 방향을 유연하게 처리하는 방식

            - grid : 격자 무늬 형식으로 화면을 분할하는 형식
