태그 위치 관련 

position 속성은 태그의 위치를 지정
        position: relative; (상대적 위치)
        - 지정된 태그 내부 다른 태그가 상대적인 위치에
        지정할 수 있도록 하는 기준 속성
        
        position : absolute; (절대적인)
        - 기본 태그의 배치 순서를 무시하고 
        지정된 절대 위치에 태그 배치

        position : fixed; (고정)
        - 항상 고정된 위치에 태그 배치

        position : sticky; (스크롤에서 많이 사용)
        - 특정 스크롤 위치에 도달시 고정시킴
        - relative + fixed 의 개념
        예) top bottom left right

        z-index : 정수
        기본값 : 0 
        수가 클수록 맨 앞으로 보임
        수가 작을수록 숫자가 맨 뒤로 보임
