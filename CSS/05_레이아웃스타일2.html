padding margin border 방향성 (외워!)

        
            000 - padding, margin, border를 나타냄
            000-top : 크기 -> 위쪽 방향 크기 지정
            000-button : 크기 -> 아래쪽 방향 크기 지정
            000-left : 크기 -> 왼쪽
            000-right : 크기 -> 오른쪽
            
            단, border 는 [크기 모양 색상]
        

        
padding margin border-width 크기 한번에 지정

        
            속성 값 1개 : 상하좌우 모두
            
            속성 값 2개 : 상하 , 좌우
            padding : 20px 10px 상하 20, 좌우 10px
            margin : 20px 5px 상하 20, 좌우 5px

            속성 값 3개 : 상 좌우 하
            margin : 30px 10x 5px 
            상30,  좌우 10px, 하5px
            
            속성 값 4개 : 상 우 하 좌
