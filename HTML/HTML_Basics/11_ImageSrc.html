이미지 태그

            -웹 문서에 사진/그림 이미지 삽입 태그
            -[속성(태그에 작성해야하는 내용)]
            1. src : 삽입할 이미지 경로 (파일위치, 인터넷 주소)
            2. width : 가로 길이 지정
            3. height : 세로 길이 지정
            4. alt : 이미지 설명 작성 > 시각 장애인 분들에게 설명 읽어주기 지원
            img src ="경로" alt="설명" width="" height=""
