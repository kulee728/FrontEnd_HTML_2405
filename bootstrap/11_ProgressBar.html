프로그래스 바

            Progress Bar : 작업의 진행 상황을 시각적으로 표시
            완료상황 기준 100%로 사용자에게 현재 작업을 얼마나 완료했는지에 대한 시각적 정보 제공
            단계가 있거나 대기 시간이 있는 작업에 유용하게쓰인다.

            속성
            progress : 프로그래스 바의 컨테이너 역할
            progress-bar : 실제로 진행 상황을 표시하는 역할 (애니메이션, 색상 등은 제공가능)
            aria-valuemin : 진행 시작 % 설정, 보통은 0%
            aria-valuemax : 진행 끝 % 설정, 보통은 100%
            aria-valuenow : 현재 진행률
            style : 프로그래스바 너비 설정
