block 형식

            블록 형식 태그 > 화면에서 공간을 세로로 분할해주는 역할
            --- h1~h6 (제목)
            --- p, pre (문단)
            --- hr (수평선)
            --- div (영역 분할) >>>> text, img, 화면에 나타내고 싶은 모든것에 다 쓴다
            블록 형식 태그들은 내용이 짧더라도 한 줄을 모두 차지하려는 성질이 있어,
            다음 태그 내용은 다음에 작성.
첫번째 DIV
두번째 DIV
세번째 DIV

div 형식

            - 가장 많이 사용하는 형식, 문서를 구분 짓는데에 사용
            - 사용시 이중 div 형태로 많이 사용
            - class는 div 뿐 아니라 모든 태그에서 사용 가능하다.
            - class는 홈페이지를 꾸밀 때 style 내부 명시로 많이 사용한다.

Inline 형식

            - 화면에서 가로를 사용 (가로 화면을 세로로 잘라서 나눈다)

            - 한 줄을 모두 사용하지 않고, 내부 작성된 크기 만큼만 차지한다.
            - width/height 크기 지정 속성 사용 불가
            - line 형식 태그는 다음과 같다.
            --- b (글씨 굵게)
            --- i (글씨 기울이기)
            --- mark (글씨 배경 하이라이팅)
            --- span (영역 분할) 등이 있다

Inline 형식은 가로에서 세로로 분할하는 성질을 갖고 있기에, 줄 넘김 없이 계속 옆으로만 작성된다.

  • Inline 형식 : 가로방향, 내용을 작성해준만큼에만 적용되는 영역

태그 textarea

            - 여러 줄을 입력할 수 있는 입력상자를 생성한다.
            - 속성 name : 제출하는 곳에서 데이터를 읽을 때 필요(변수 매핑)
            - rows : textarea 행 크기 지정 (rows="3" >> 3행)
            - cols : textarea 열 크기 지정 (cols="3" >> 3행)
            - style : resize:none > 크기 변경 불가
